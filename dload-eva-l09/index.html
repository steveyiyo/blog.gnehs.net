華為P9救磚(EVA-L09)

勝勝 · 2017 年 02 月 28 日

標題上打EVA-L09是因為文章裡的檔案只支援這個型號啦，事實上教學應該華為的裝置都通用 若因本教學造成手機損毀恕不負責 首先，先把檔案下載下來

載點:dload-huawei-p9-eval09-eva-b136.zip 原址:Stock ROM dload file - P9 EVA-L09 B136

接著，把壓縮檔內的"dload"資料夾整個放進你的SD卡根目錄並將SD卡放回手機 同時按下"電源鍵"&“音量+"&“音量-” 接著就會進入 eRecovery 模式，手機會自己開始刷機 出現 OK 就好了!
